# LLM 文本处理器

> 基于 LLM 的智能文本处理工具

## 功能介绍

### 1️⃣ AI 摘要
- 输入原文后，点击「生成摘要」按钮
- **仅显示摘要，不替换原文**
- 摘要内容展示在独立区域

### 2️⃣ 智能分类
- 点击「智能分类」按钮
- AI 自动分析文本，生成 3-5 个分类标签
- **点击分类标签 → 跳转到对应段落**

### 3️⃣ 最终内容
- 与第二块同步滚动
- **可直接编辑内容**
- 支持与第二块内容对比

### 4️⃣ 导出
- 点击「导出」按钮
- 下载为 TXT 文件

## 支持的服务商

| 服务商 | 模型 |
|--------|------|
| DeepSeek | deepseek-chat |
| OpenAI | gpt-4, gpt-3.5-turbo |
| Moonshot | moonshot-v1-8k |
| 智谱 AI | glm-4 |
| Ollama | llama3 |

---

## 配置步骤

1. 点击右上角 ⚙️ 配置
2. 选择服务商
3. 填写 API Key
4. 选择模型
5. 点「测试」验证
6. 点「保存」

---

## 使用流程

```
输入原文 → 生成摘要 → 智能分类 → 编辑导出
```
